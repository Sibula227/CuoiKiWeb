### Thang trình độ năng lực (TĐNL)

- **0.0 ≤ TĐNL ≤ 1.0 (Cơ bản):** Nhớ: ghi nhớ/ nhận ra/ nhớ lại kiến thức.
- **1.0 < TĐNL ≤ 2.0 (Đạt yêu cầu):** Hiểu: tự kiến tạo kiến thức, giải thích, phân loại, minh họa.
- **2.0 < TĐNL ≤ 3.0 (Đạt yêu cầu):** Áp dụng: thực hiện/ áp dụng kiến thức tạo ra sản phẩm.
- **3.0 < TĐNL ≤ 4.0 (Thành thạo):** Phân tích: phân tích tài liệu/ kiến thức thành các chi tiết và chỉ ra mối quan hệ.
- **4.0 < TĐNL ≤ 5.0 (Thành thạo):** Đánh giá: đưa ra nhận định, dự báo theo tiêu chuẩn.
- **5.0 < TĐNL ≤ 6.0 (Xuất sắc):** Sáng tạo: kiến tạo/ sắp xếp/ thiết kế/ khái quát hóa tạo ra cấu trúc mới.

### Cơ sở vật chất phục vụ học tập

- **Phòng máy tính:** Máy tính cài phần mềm thiết kế, xử lý file, bình trang điện tử, mô phỏng in.
- **Xưởng phơi bản:** Máy xuất phim, máy phơi bản offset/flexo, máy chế tạo bản in lưới, máy đo bản.
- **Phòng CTP:** Hệ thống chu trình làm việc CTP, thiết bị ghi/hiện bản, máy in phun khổ lớn, máy đo màu quang phổ.
- **Xưởng In offset:** Máy in Offset 1, 2, 4 màu; máy cắt giấy; máy pha mực.
- **Phòng thí nghiệm bao bì:** Phần mềm thiết kế cấu trúc hộp ArtiosCAD, thiết bị làm mẫu hộp không khuôn, máy bồi carton, máy thử độ bền nén.
- **Xưởng thành phẩm:** Máy gấp sách, đóng kim, khâu chỉ, vào bìa keo, ép nhũ, dán màng, máy bế.
- **Phòng thí nghiệm vật liệu in:** Máy thử độ bền giấy, máy đo độ nhớt mực, máy đo độ mịn hạt mực, thiết bị đo độ dày, đo pH, đo độ cứng cao su, kính hiển vi soi bề mặt.
